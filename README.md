# PersonalSkin — פרויקט Python (Flask)

אפליקציית ווב להתאמת שגרת טיפוח אישית: הרשמה והתחברות, שאלון פרופיל עור, יצירת שגרת בוקר/ערב (בינה מלאכותית או מצב גיבוי), ודף קהילה לשיתוף פרופיל ושגרה.

## דרישות

- Python 3.10+ (נבדק עם 3.12)
- חבילות: ראו `requirements.txt`

## התקנה והרצה

```text
cd personal_skin
py -3 -m pip install -r requirements.txt
py -3 run.py
```

פתחי בדפדפן: `http://127.0.0.1:5000`

## הגדרות (אופציונלי)

1. העתיקי `.env.example` לקובץ `.env`.
2. `FLASK_SECRET_KEY` — מחרוזת אקראית לסשן (מומלץ בייצור).
3. `OPENAI_API_KEY` — ליצירת שגרה עם מודל שפה. בלי מפתח, המערכת משתמשת בשגרה לפי כללים (גיבוי).

## מבנה הפרויקט

- `run.py` — נקודת כניסה.
- `app/` — יישום Flask (מודלים, בלופרינטים, שירות AI).
- `templates/` — תבניות HTML (עברית, RTL).
- `static/css/` — עיצוב.
- קובץ SQLite `personal_skin.db` נוצר אוטומטית בספריית ההרצה.

## הערה משפטית

ההמלצות מיועדות להסבר כללי בלבד ואינן תחליף לייעוץ רפואי או מקצועי.
